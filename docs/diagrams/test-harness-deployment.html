Test Harness Deployment

Three test paths: Unit (fast) → Integration (docker-compose) → K8s e2e (Kind + Knative)

Fast Path (default)

  • • make test → unit tests (~15s)
  • • make test-integration → docker-compose Fluss
  • • make test-e2e → full data plane pipeline
  • • Total: ~90s for full build
  • • Requires: Docker running

Infrastructure

  • • LocalStack 4.6.0 for S3-compatible storage
  • • WireMock in-process (no container)
  • • Lakehouse: +Polaris + Flink
